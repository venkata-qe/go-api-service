# GO API Service

GO API Service is a rest API service which has movie endpoints.

## Installation
Install Go Lang

## Usage

### Start mongod container in docker
```
docker run -d -p 27017-27019:27017-27019 --name mongodb mongo:4.0.4
```

### Start the api service
```
go run app.go
```
